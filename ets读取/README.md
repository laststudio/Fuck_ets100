# ETS100 读取文档索引

这里集中整理 Fe 项目中与 ETS100 读取、解析、登录相关的开发文档。按职责拆成五类，避免答案解析、本地读取、云端接口和企业登录链路互相混在一起。

| 文档 | 说明 |
|------|------|
| [答案解析方法](./ets100答案解析方法.md) | 说明拿到 `content.json` 后，如何按地区、题型和 `structure_type` 提取题目、答案、选项与朗读原文。 |
| [本地模式读取逻辑](./本地模式读取逻辑.md) | 说明本地 ETS100 目录结构、resource 扫描、时间分组、`res.json` 顺序判定、`data/sectionData` 兜底和 assets 映射。 |
| [云端 API 与跨平台实现](./API_DOC.md) | 说明 ETS100 云端接口协议、签名、登录、设备绑定、作业列表、ZIP 下载解压和跨平台实现建议。 |
| [云端模式逻辑](./云端模式逻辑文档.md) | 说明当前 Android App 内云端模式从登录、保存认证、加载作业、下载解析到页面展示的完整业务流程。 |
| [企业登录（畅言）](./畅言网页登录.md) | 说明畅言通行证网页登录、`checkLogin` 拦截、`/user/login-zt` 换取 token、多账号选择与云端读取衔接。 |

## 阅读顺序

1. 只关心题目/答案怎么解析：先看 [答案解析方法](./ets100答案解析方法.md)。
2. 只关心本地直读、Shizuku、Root 为什么能找到试卷：先看 [本地模式读取逻辑](./本地模式读取逻辑.md)。
3. 想重写云端读取脚本或跨平台客户端：先看 [云端 API 与跨平台实现](./API_DOC.md)。
4. 想理解 App 里云端模式怎么串起来：先看 [云端模式逻辑](./云端模式逻辑文档.md)。
5. 想理解“讯飞登录”/企业账号网页登录：先看 [企业登录（畅言）](./畅言网页登录.md)。
